Seeed Studio mmWave Kit - MR24HPC1

  1. Connect the kit to your computer via a USB-C type cable.
  2. Click the Connect button and select the port number device that has a JTAG name.
  3. Flash Firmware. Simply click on INSTALL SEEED STUDIO MMWAVE KIT.
  4. After flashing the firmware, you need to configure the network for the kit.
    Using your computer or mobile phone, search for a network named seedstudio-mr24hpc1
    and configure your 2.4GHz network in the 192.168.4.1 page.
  5. Enjoy your trip to Home Assistant!
